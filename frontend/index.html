Welcome to Academic Performance Prediction System

This system uses machine learning to predict student academic performance and identify at-risk students.

  • Lecturer Module: Input assessment scores and attendance records
  • AI Prediction: Logistic Regression model to predict student risk levels
  • Dashboard: View risk statistics and identify high-risk students
